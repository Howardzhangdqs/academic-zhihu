# 知乎过滤器（Academic Zhihu Filter）

让你的知乎变得更学术。使用大语言模型（LLM）自动识别和过滤知乎信息流中的非学术内容，帮你聚焦真正有价值的学术讨论。

## 功能特性
- **智能分类**：通过大语言模型对知乎首页信息流中的问题和回答进行学术/非学术内容分类
- **灵活过滤**：支持三种隐藏模式 —— 颜色变淡、折叠、折叠+变淡，被过滤的内容鼠标悬停即可查看
- **自定义提示词**：可编辑系统提示词，定义你自己的"学术"标准，筛选符合你兴趣的内容
- **多模型支持**：支持 OpenAI 兼容接口和 Anthropic 接口，可自定义 Base URL、API Key 和模型名称，兼容 DeepSeek、智谱 GLM、OpenRouter 等多种模型服务
- **高级配置**：支持自定义请求头和请求体，满足各种 API 代理需求
- **一键测试**：内置连通性测试，配置后可快速验证 API 是否可用
- **左下角悬浮按钮**：随时开关过滤功能和调整设置，不影响正常浏览

## 使用方法
1. 安装脚本后打开知乎首页
2. 点击左下角悬浮按钮打开设置面板
3. 填入你的 API Key（支持 OpenAI Compatible 和 Anthropic Compatible 接口）
4. 根据需要调整提示词和过滤模式
5. 保存设置后，脚本将自动过滤信息流中的非学术内容

## 隐私说明
- 你的 API Key 仅存储在浏览器本地（油猴 GM 存储），不会上传到任何第三方服务器
- 脚本仅将知乎信息流中的标题和摘要文本发送到你配置的 LLM API 进行分类，不发送任何其他数据
